# uavcan v0.1 fake gps publisher

## Preparation

1. Clone repo

2. Install python packages:
```
pip3 install -r requirements.txt
```

3. Make scripts as executable
```
chmod +x run.sh
chmod +x fake_gps.py
```

## How to run

1. Insert a sniffer device to your computer
2. Then type:
```
./run.sh
```
